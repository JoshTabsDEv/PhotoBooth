## <img src="https://em-content.zobj.net/source/apple/391/thinking-face_1f914.png" width="25"/> What Is This?

You know those photo booths at the mall where you squeeze in with your friends, the curtain barely closes, and you somehow spend $10 for 4 tiny pictures?

**Yeah, this is that. But free. And in your browser. And you don't have to share a tiny bench with anyone.**

Photo Booth is a web app that turns your phone or laptop camera into a full-blown photo booth experience -- complete with timers, photo strips, stickers, text overlays, and templates. Take photos, make them look awesome, download them. That's it. No sign-up, no subscriptions, no "please enter your credit card for the free trial."

## <img src="https://em-content.zobj.net/source/apple/391/hammer-and-wrench_1f6e0-fe0f.png" width="25"/> Tech Stack

> *"We chose the best tools so you don't have to think about them."*

| Tech | Why? |
|---|---|
| **Next.js 15** | Because SEO matters and server components are chef's kiss |
| **React 19** | It's React. It reacts to things. Groundbreaking. |
| **TypeScript** | So we catch bugs before they catch us |
| **Tailwind CSS v4** | Writing CSS is overrated. Utility classes go brrrr |
| **Konva.js** | Canvas manipulation that doesn't make you cry |
| **react-konva** | Konva but make it declarative JSX |
| **Zustand** | State management that weighs less than your favicon |
| **shadcn/ui** | Beautiful components. Zero runtime cost. Free therapy. |
| **Vercel** | Deploy on push. It just works. |

## <img src="https://em-content.zobj.net/source/apple/391/package_1f4e6.png" width="25"/> Getting Started

```bash
# Clone the repo (you know the drill)
git clone https://github.com/JoshTabsDEv/PhotoBooth.git
cd PhotoBooth

# Install dependencies (grab a snack, it'll be a sec)
npm install

# Fire it up!
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) and prepare to be amazed. Or at least mildly entertained.

## <img src="https://em-content.zobj.net/source/apple/391/file-folder_1f4c1.png" width="25"/> Project Structure

```
photo-booth/
├── public/
│   ├── stickers/          # The fun stuff
│   ├── templates/         # Photo layouts & frames
│   └── images/            # OG images, landing assets
├── src/
│   ├── app/               # Next.js pages (landing, booth, editor, preview)
│   ├── components/        # React components organized by feature
│   ├── hooks/             # useCamera, useTimer, useCanvas (the real MVPs)
│   ├── lib/               # Canvas utils, camera helpers, constants
│   ├── store/             # Zustand stores (booth + editor state)
│   └── types/             # TypeScript types (because we're civilized)
└── package.json           # The dependency list that keeps growing
```
